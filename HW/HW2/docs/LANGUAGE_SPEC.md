# VoxLang 語言設計規範

## 1. 語言概述

**語言名稱**: VoxLang (Vox Programming Language)

**設計目標**: 教學用類 C 語言，編譯到自訂位元組碼

**核心特性**:
- 強類型語言
- 編譯到位元組碼 + 虛擬機器執行
- 堆疊式虛擬機器
- 支援基本資料類型、函式、陣列、控制流

## 2. 資料類型

| 類型 | 描述 | 範例 |
|------|------|------|
| int | 32 位元整數 | 42, -10, 0 |
| float | 64 位元浮點數 | 3.14, -2.5 |
| bool | 布林值 | true, false |
| string | 字串 | "Hello World" |
| array | 陣列 | [1, 2, 3] |

## 3. 語法特性

### 3.1 變數宣告
```
var x: int = 10;
var name: string = "Vox";
var arr: array = [1, 2, 3];
```

### 3.2 函式定義
```
func add(a: int, b: int) -> int {
    return a + b;
}
```

### 3.3 控制流
- 條件: if (condition) { ... } else { ... }
- 迴圈: while (condition) { ... }
- for 迴圈: for (var i: int = 0; i < 10; i = i + 1) { ... }

### 3.4 運算子
- 算術: +, -, *, /, %
- 關係: ==, !=, <, >, <=, >=
- 邏輯: &&, ||, !

## 4. 內建函式

- print(value): 輸出值
- input(): 取得使用者輸入
- len(arr): 取得陣列長度

## 5. 編譯器架構

1. **詞法分析器 (Lexer)**: 原始碼 -> Token 流
2. **語法分析器 (Parser)**: Token 流 -> AST
3. **語意分析**: 類型檢查、作用域分析
4. **位元組碼生成器**: AST -> 位元組碼
5. **虛擬機器 (VM)**: 執行位元組碼

## 6. 位元組碼設計

採用基於堆疊的指令集，支援:
- 載入/儲存指令
- 算術運算指令
- 控制流指令
- 函式呼叫指令
- 陣列操作指令
